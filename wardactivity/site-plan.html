Overview

Purpose

The purpose of the website is to help the ward members to have information about future activities. My calling in my ward is Ward Activities Committee Member. It is another tool for them to know and stay informed about the activities that are happening in the ward. Also, it would be great for the members to see past events on the website, so they can see the great memories made and want to be more apart of. This project will help me to magnify my calling.

Audience

The audience for this website is the members of the C5 Ward.

Dynamic elements

JavaScript will help out by providing interactivity and enhancing the user experience on the website. I would like to have a calendar that shows upcoming activities with a countdown to the next activity. I would like to have a video/photos slideshow of past activities. I would like to give the user the choice to toggle between light and dark mode.
